JAVASCRIPT CÓ THỂ LÀM GÌ?

JavaScript được tạo ra với mục đích xử lý các tác vụ phía Client như

  1. Tạo menu xổ xuống. Các item menu khi đưa chuột vào sẽ xổ xuống các mục con của nó.
  2. Thay đổi nội dung trang web. Ví dụ: hiện kết quả khen/chê tùy user trả lời câu hỏi đúng dau, hiện lời chào tùy thời gian buổi sáng hay chiều…
  3. Thêm các thành phần động vào trang web như slide show, carousel.
  4. Tạo các hiệu ứng cho hình ảnh, text…trong trang. Ví dụ các hiệu ứng ẩn hiện hình, đổi hình khi đưa chuột vào.
  5. JavaScript có thể dùng viết game

    https://www.w3schools.com/graphics/game_intro.asp
